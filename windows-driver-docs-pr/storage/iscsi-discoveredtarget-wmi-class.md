# <a name="iscsidiscoveredtarget-wmi-class"></a>ISCSI\_DiscoveredTarget WMI 类


## <span id="ddk_iscsi_discoveredtarget_wmi_class_kr"></span><span id="DDK_ISCSI_DISCOVEREDTARGET_WMI_CLASS_KR"></span>


ISCSI\_DiscoveredTarget WMI 类提供与搜索到的目标相关的信息。

此类中定义，则不发布*Discover.mof*。

当 WMI 工具套件编译此类定义时，它会生成[ **ISCSI\_DiscoveredTarget** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsifnd/ns-iscsifnd-_iscsi_discoveredtarget)数据结构
