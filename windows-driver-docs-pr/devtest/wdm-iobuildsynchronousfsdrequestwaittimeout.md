# <a name="iobuildsynchronousfsdrequestwaittimeout-rule-wdm"></a>IoBuildSynchronousFsdRequestWaitTimeout 规则（wdm）


如果**IoBuildSynchronousFsdRequestWaitTimeout**规则检测到此驱动程序将无限期等待，直到降低驱动程序返回，因为 IRP 的事件需要在完成例程中发出信号。

|              |     |
|--------------|-----|
| 驱动程序模型 | WDM |
