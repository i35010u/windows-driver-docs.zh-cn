# <a name="fileobjectconfigured-rule-kmdf"></a>FileObjectConfigured 规则（kmdf）


**FileObjectConfigured**规则指定对[**WdfRequestGetFileObject**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdfrequest/nf-wdfrequest-wdfrequestgetfileobject)方法的调用，前面是对[**WdfDeviceInitSetFileObjectConfig**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdfdevice/nf-wdfdevice-wdfdeviceinitsetfileobjectconfig)的调用。

|--------------|------|
