KSMETHODSETID\_StreamAllocator 方法集指定用于基于 IRP 的分配器接口函数。

该方法设置包括：

[**KSMETHOD\_STREAMALLOCATOR\_分配**](ksmethod-streamallocator-alloc.md)

[**KSMETHOD\_STREAMALLOCATOR\_免费**](ksmethod-streamallocator-free.md)
