# <a name="print-ticket-and-print-capabilities-technologies"></a>打印票证和打印功能技术


Microsoft Windows Vista 引入了打印功能和打印票证技术。 与以前的方法相比，这些新技术可提供更灵活、更具扩展性的打印配置和打印文档的外观。 Microsoft Windows Presentation Foundation （WPF）和 Microsoft Win32 应用程序、XPSDrv 打印驱动程序和基于图形设备接口（GDI）的打印驱动程序都可以使用这些技术。
