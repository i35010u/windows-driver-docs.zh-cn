# <a name="debugging-resources"></a>调试资源

使用 Windows 调试工具可以调试 Windows 系统上的驱动程序、应用程序和服务。 该工具集内的核心调试引擎称为“Windows 调试器”。 你可以使用传统接口（WinDbg、CDB 和 NTSD），这些接口包括在适用于 Windows 的调试工具中。
