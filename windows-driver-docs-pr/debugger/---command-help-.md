# <a name="-command-help"></a>? （命令帮助）


问号 (**？**) 字符显示所有命令和运算符的列表。

**请注意**  本身问号显示命令的帮助。 [ **？ 表达式**](---evaluate-expression-.md)语法计算给定的表达式的值。

```dbgcmd
    ?
```

-------
